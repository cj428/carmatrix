Pasos para ejecutar el programa de desplazamiento:

1. Importar la carpeta con el proyecto a NetBeans 8.6
2. ingresar a carmatrix/Source Package/carmatrix/camatrix.java y dar click derecho seleccionado "run file"
3. ingresar la posicion maxima en X (filas)
4. ingresar la posicion maxima en y (columnas)
5. Ingresar los valores de desplazamiento con el formato:
	Desplazamiento, Dirección
	Nota: si va a ingresar varios desplazamientos, debe separarlos por ;
6. La consola le mostrara los datos ingresados, y el desplazamiento que se generó
